# Social Media Prompt Generator

Generate scroll-stopping social media content across all major platforms using battle-tested ChatGPT prompts.

## When to use this skill

Use when a user needs:
- LinkedIn thought leadership posts
- Twitter/X threads and viral tweets
- Instagram captions and carousels
- TikTok script hooks
- Hashtag strategies
- Content calendar planning
- Cross-platform content repurposing

## Core prompts

### LinkedIn Growth

Ask the user for their industry and topic, then use:

```
Act as a LinkedIn thought leader in [industry]. Write a post sharing a contrarian opinion about [common belief]. Start with: "Everyone says [common belief]. But here's what they're missing."
```

### Twitter/X Threads

```
Act as a Twitter growth strategist. Write a 10-tweet thread about [topic]. Tweet 1: scroll-stopping hook. Tweets 2-8: one insight per tweet. Tweet 9: most valuable insight. Tweet 10: CTA to follow + retweet.
```

### Instagram Carousels

```
Act as an Instagram content designer. Turn this into a 10-slide carousel script. For each slide: headline (under 10 words), body text (under 30 words), design suggestion.
```

### TikTok/Reels Scripts

```
Act as a TikTok content creator. Write a 60-second video script. First sentence is the hook. Mark visual cues. Include [pause] and [emphasis] cues. End with CTA.
```

### Engagement Questions

Generate audience engagement:

```
Generate 5 engagement questions about [topic] designed to get comments and shares.
```

## Hook library (50 hooks)

### Curiosity (10)
- "This one trick changed everything."
- "Nobody talks about this."
- "Here's what they don't tell you about [topic]."
- "I tried [method] for [time]. Here's the truth."

### Contrarian (10)
- "[Popular advice] is wrong. Here's why."
- "You don't need [popular tool]. You need [unexpected thing]."
- "I quit [popular method]. Best decision I ever made."

### How-To / List (10)
- "[Number] things I wish I knew about [topic]."
- "The [number]-step framework for [desired outcome]."
- "How I [achieved metric] in [time]."

### Question (10)
- "What would happen if [unexpected scenario]?"
- "Are you making this [topic] mistake?"
- "What if I told you [surprising statement]?"

### Relatable (10)
- "I almost quit [effort]. Here's why I didn't."
- "Real talk: [honest opinion]."
- "I failed at [X] times before I got it right."

```
## Platform: [LinkedIn/Twitter/Instagram/TikTok]
### Post Type: [thread/carousel/video/caption]
### Hook: [the hook]
### Body: [content]
### CTA: [call to action]

```
