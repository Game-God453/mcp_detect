# Portrait & Photography Patterns

## 什么时候用

- 生成人物肖像 / 编辑级写真 / 街拍感 / 偶像风格
- 需要强光环境感 / 具体相机型号感 / 特定年代氛围

## 六大子类型速查

### A. 便利店霓虹 / 夜店街拍（35mm flash editorial）
**关键词组合**：
```
35mm film photography, harsh fluorescent/neon lighting,
authentic film grain, high contrast, slight color cast,
cinematic street editorial style, intimate medium shot
```
**光线要诀**：冷光源（荧光管/LED）+ 暖霓虹（粉蓝红）混合，产生强对比
**典型场景**：便利店玻璃门、地铁站台、深夜巷口、夜店外

### B. 温泉旅馆 / 和室低光（Japanese onsen style）
**关键词组合**：
```
Japanese onsen ryokan interior, soft warm tungsten lighting,
tatami mat ambiance, shoji paper screen diffused light,
Fujifilm Pro 400H film emulation, warm color grading
```
**光线要诀**：单一暖光源（暖白/橘黄），纸屏/布帘柔化
**典型场景**：榻榻米房间、浴池边缘、和室走廊

### C. 偶像 CCD 抓拍（Korean idol CCD flash）
**关键词组合**：
```
CCD digital camera on-camera flash, harsh flat flash lighting,
2000s Y2K aesthetic, overexposed highlights, slight red-eye,
amateur candid composition, raw un-retouched feel
```
**光线要诀**：正面直闪 → 脸部扁平 + 背景漆黑 + 阴影硬边
**典型场景**：后台抓拍、KTV包厢、生日派对

### D. 35mm 柔雾编辑（soft black mist editorial）
**关键词组合**：
```
35mm film with black mist filter 1/4 strength,
soft bloom on highlights, creamy skin rendering,
Kodak Portra 400 film emulation, editorial fashion shot,
shallow depth of field f/1.4 subject separation
```
**光线要诀**：单侧主光 + 补光，雾镜让高光发散
**典型场景**：白墙影棚、酒店房间、车窗边

### E. 手机自拍镜（mirror selfie bedroom）
**关键词组合**：
```
iPhone mirror selfie, bedroom vanity with LED ring light,
slight lens distortion, natural amateur composition,
authentic grainy phone photography quality,
handheld holding phone visible in reflection
```
**光线要诀**：环形灯 / 梳妆镜灯 / 窗户自然光
**典型场景**：卧室梳妆台、健身房更衣间、酒店

### F. 9:16 竖屏偶像出街
**关键词组合**：
```
9:16 vertical portrait aspect ratio for social media,
candid outdoor paparazzi style, motion blur on background,
direct sunlight with hard shadows, fashion street shot,
full body composition with head room
```
**光线要诀**：自然光 / 黄金时刻 / 硬阴影感
**典型场景**：街拍 / 商场门口 / 片场休息

## 解剖学词汇库（拿来即用）

### 面部
- **眼形**：almond-shaped, fox-eyed, hooded, doe-eyed, cat-eyed, wide-set
- **眼皮**：natural double eyelids, inner double lids, monolids
- **鼻**：high nose bridge, refined narrow nose, aquiline, button nose
- **下巴**：V-shaped jawline, soft heart-shaped face, angular square jaw
- **颧骨**：high defined cheekbones, soft rounded cheeks
- **唇**：glossy natural pink lips, matte mauve, slightly parted, cupid's bow

### 皮肤
- **色调**：cool ivory undertone, warm honey undertone, neutral beige
- **质感**：porcelain skin with specular highlights, dewy with natural oil,
         matte skin with visible pores, airbrushed editorial
- **瑕疵**：subtle freckles across nose, natural beauty mark, light acne marks

### 发型
- **长度**：long straight, bob cut, pixie, shoulder-length layered
- **状态**：messy high ponytail with loose strands, sleek low bun,
          wet hair strand effect, bed head tousled, air-dried wavy
- **颜色**：jet black, dark chocolate brown, platinum blonde, strawberry blonde

### 妆容
- **风格**：natural dewy makeup, smoky eye editorial, glass skin Korean,
          no-makeup makeup look, dramatic stage makeup
- **重点**：glossy lips, winged eyeliner, flushed cheeks, highlighter on cheekbones

### 便利店夜拍（最经典）
```
35mm film photography with harsh convenience store fluorescent lighting
mixed with colorful neon signs from outside, authentic film grain,
high contrast, slight color cast, cinematic street editorial style,
intimate medium shot,
[AGE] [ETHNICITY] [GENDER] with [FACIAL FEATURES], [EYE DESCRIPTION],
[NOSE], [JAWLINE], [SKIN TONE AND QUALITY],
[HAIR LENGTH/STYLE/COLOR], wearing [CLOTHING],
[POSE] against [ENVIRONMENT],
[EXPRESSION/GAZE DESCRIPTION],
bright cold fluorescent store light from inside mixed with [COLORED NEON]
glow from outside signs, realistic reflections on glass door,
blurred store interior with shelves and snacks in background,

```

## 反面清单（避免）

❌ "beautiful young woman" → 没有信息量
✅ "early 20s Chinese female with almond-shaped fox eyes and V-shaped jawline"

❌ "good lighting" / "cinematic lighting" → 太泛
✅ "harsh top-down fluorescent mixed with warm pink neon from right"

❌ "wearing nice clothes" → 无视觉
✅ "oversized white button-up shirt unbuttoned at top, tied at waist"

❌ 只说"sexy" / "cute" → 模型产出刻板
✅ "seductive playful yet slightly vulnerable gaze with quiet temptation"

## 项目溯源

### AI 视频真人感 JSON Prompt（2026-04-28，@idall.ai 对标）

**原需求**：Neal 贴了一段 Taiwanese 早餐店真实感 prompt 作为参考风格，要求以同等真实感改写两段 AI 视频（写实系 IG 账号 @idall.ai 的典型输出：一段是男性白衬衫室内近景，一段是女性极端面部特写），产出 JSON 结构。

**目标**：让 AI 视频模型（Sora/Veo/Seedance/Kling）生成出"看起来不像 AI"的真人感肖像视频。

**关键决策（写实感三支柱）**：

1. **具象光源命名**：禁用 `good lighting` / `cinematic lighting`，改写成「相机哪侧的什么窗、什么色温、catchlight 落在眼球哪一象限」。例：`large north-facing window from camera-left, 5200K slightly cool` + `single soft rectangular catchlight in upper portion of both eyes`。

2. **瑕疵清单（Imperfection Checklist）——写实感的通行证**：
   - 皮肤：`visible pores` / `fine crow's feet` / `slight redness on cheeks` / `sun damage` / `faint under-eye vein on shadow side`
   - 五官不对称：`asymmetric eye size (left slightly smaller, human baseline)` / `asymmetric micro-expressions (one corner of mouth moves first)`
   - 标志性特征：`tiny mole above upper lip on left side` / `single fine baby-hair at hairline`
   - 拒绝 AI 常见缺陷（写进 negative_prompt）：plastic skin / airbrushed / symmetrical CGI face / doll-like eyes

   **原则**：写"完美"只会得到假。写"具体的不完美"才是真。

3. **时间轴动作（`action_beats` 字段）——视频独有**：
   把 6-10 秒切成 2-3 个时间锚点，每个锚点描述 1-2 个微动作。禁用 `she talks` / `he smiles`，改写成：
   ```
   "0.0-2.0s: settled neutral gaze, one slow blink",
   "2.0-4.0s: eyes flick 3° screen-left then return, pupil dilation shift",
   "4.0-6.0s: lips part 1mm, small inhale, gaze holds"
   ```
   角度（3°）、距离（1mm）、节奏（every 2s）——这些具体单位把时序控制从"全靠模型猜"降到"按拍子走"。

**JSON Schema（真人感视频模板）**：
```
{
  shot: { type, aspect_ratio, duration, camera, lens, frame_rate, styling }
  subject: { identity, ethnicity, build, hair, facial_hair, skin, eyes, expression, wardrobe, accessories }
  action_beats: [ 时间锚点数组 ]
  environment: { location, background, left_side/right_side, floor, props }
  lighting: { key, fill, rim, catchlight, contrast, color_temperature }
  color_grade: "..."
  audio: "..."
  overlays: { watermark, ... }
  realism_anchors: [ 瑕疵清单数组 ]
  negative_prompt: "..."
}
```

**踩过的坑**：
- **不要直接写明星名字**。训练数据污染 + 平台审核，改写成 `Ana-de-Armas-adjacent features (do not name a real celebrity in final prompt)` 让模型理解参考人种/骨相、又不触发版权。
- **肤色别写 `beautiful skin`**。改成 `warm olive undertone, dewy but not oily` + 具体泛红区域。
- **catchlight 必须命名形状**。`rectangular catchlight`（说明光源是窗户）vs `round catchlight`（说明是灯泡）——模型会按 catchlight 形状反推光源属性。
- **`realism_anchors` 和 `negative_prompt` 要对着写**。正面写"保留 pore detail"，负面就写"no plastic skin"——一正一反双重约束才稳。

**学到的**：
- **AI 视频真人感 = 瑕疵 + 具象光源 + 时间轴微动作**，缺一个就"糊"掉一层。
- **JSON 结构对视频 prompt 比对静态图更关键**：因为多了 `action_beats` 和 `audio` 两层时间维度，平铺描述塞不下。
- **早餐店 prompt 之所以真实感强**，是因为它在前景物品（杯子上的卡通印花+中文小字、橙色印刷纸袋）和背景细节（街景+机车+白车、菜单板的蓝红配色）里堆满了具体物件——不是 `a cafe`，是"一排能点名的东西"。同样思路搬到人脸特写，就变成「把具体瑕疵当道具」。

**可迁移场景**：
- 任何 img2video / text2video 的肖像镜头
- IG/TikTok 伪真人账号风格（@idall.ai / @emilyai01 这类）
- 采访/口播类视频的首帧控制
- 替代 `photorealistic` / `hyperrealistic` 这类空词
